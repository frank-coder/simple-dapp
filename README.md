# simple-dapp

A test dapp that connects to multichain wallets via walletconnect and wagmi
Also sends testnet BUSD to any specified address

install packages
 - yarn
```yarn```
 - npm
```npm -i```

development 
```yarn dev```

Tech stack/packages
 - React
 - Typescript
 - Vite
 - Web3Modal
 - Wagmi
